# HashMap

HashMap data structure implementation.

## Skills demonstrated

- Hashing a key for its hash code.

- Setting up buckets — array with linked list inside

- Storing key value pair in buckets.

- Updating the value associated with the key.

- Handling collisions by adding a new node.

- Growing the hash map capacity when overloaded to reduce collisions.

- Retrieving value associated with key.

- Checking for key existence in the hash map.

- Removing key from hash map.

- Clearing all key value pair entries from hash map.

- Returning all the keys, values, and entries in the hash map.
